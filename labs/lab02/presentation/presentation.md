# Цель работы

## Основная цель

Получить представление о работе с учётными записями пользователей и группами пользователей в ОС типа Linux и закрепить практические навыки их администрирования.

# Ход выполнения работы

## Переключение учётных записей

![Определение пользователя и его групп](Screenshot_1.png){ #fig:001 width=70% height=70% }

Определение активного пользователя и его групп (`whoami`, `id`).  
Переход в режим суперпользователя `root` и возврат к обычному пользователю.

## Просмотр и настройка sudoers

![Файл sudoers, открытый через visudo](Screenshot_2.png){ #fig:002 width=70% height=70% }

Безопасное редактирование `/etc/sudoers` через `visudo`.  
Анализ записи `%wheel ALL=(ALL) ALL` и роли группы `wheel` для выдачи прав через `sudo`.

## Создание пользователей alice и bob

![Создание пользователей alice и bob](Screenshot_3.png){ #fig:003 width=70% height=70% }

Создание пользователя `alice`, включение его в группу `wheel`.  
Создание пользователя `bob`, назначение паролей и проверка параметров учётных записей.

## Настройка параметров создания пользователей

![Редактирование файла /etc/login.defs](Screenshot_4.png){ #fig:004 width=70% height=70% }

Изменение общесистемных настроек в `/etc/login.defs`:

Настройка шаблонов в `/etc/skel` (каталоги `Documents`, `Pictures`, файлы и переменная `EDITOR`).

## Шаблоны окружения в /etc/skel

![Настройка EDITOR и .bashrc](Screenshot_5.png){ #fig:005 width=70% height=70% }

Добавление `export EDITOR=/usr/bin/vim` в `.bashrc` из `/etc/skel`.  
Формирование единых настроек окружения для всех новых пользователей.

## Создание учётной записи carol

Создание пользователя `carol` с использованием новых настроек.  
Проверка основной группы `users` и наличия каталогов `Documents` и `Pictures` в домашнем каталоге.

## Работа с /etc/shadow и параметрами пароля

![Просмотр и изменение записи carol в /etc/shadow](Screenshot_7.png){ #fig:007 width=70% height=70% }

## Управление группами пользователей

# Итоги работы

## Вывод

В ходе лабораторной работы освоены приёмы администрирования пользователей и групп в Linux.  
Рассмотрены механизмы выдачи привилегий через `sudo`, создание и настройка учётных записей, управление паролями и групповой принадлежностью.  
Полученные навыки являются базовыми для дальнейшего изучения системного администрирования в Unix-подобных системах.
