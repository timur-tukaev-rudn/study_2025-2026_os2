# Цель работы

## Основная цель

Получить навыки работы с утилитами управления модулями ядра операционной системы Linux.

# Ход выполнения работы

## Управление модулями ядра

![Просмотр устройств и модулей ядра](Screenshot_1.png){ #fig:001 width=70% height=70% }

Просмотр устройств с помощью `lspci -k` и определение связанных с ними драйверов.

## Просмотр и загрузка модулей

![Просмотр загруженных модулей](Screenshot_2.png){ #fig:002 width=70% height=70% }

Просмотр списка загруженных модулей с помощью `lsmod` и загрузка `ext4` через `modprobe ext4`.

## Проверка и информация о модуле ext4

![Информация о модуле ext4](Screenshot_3.png){ #fig:003 width=70% height=70% }

Получение сведений о модуле `ext4` с помощью `modinfo`: лицензия, зависимости, путь и параметры.

## Выгрузка модулей ядра

![Выгрузка модулей ext4 и xfs](Screenshot_4.png){ #fig:004 width=70% height=70% }

Попытка выгрузить модули `ext4` и `xfs`; модуль `xfs` не выгружен, так как используется файловой системой.

## Работа с модулем bluetooth

![Информация о модуле bluetooth](Screenshot_5.png){ #fig:005 width=70% height=70% }

Загрузка и выгрузка модуля `bluetooth`, просмотр зависимостей и параметров (`disable_esco`, `disable_ertm`, `enable_ecred`).

## Обновление ядра системы

![Обновление ядра и системы](Screenshot_8.png){ #fig:008 width=70% height=70% }

Обновление ядра через `dnf update kernel`, перезагрузка и проверка новой версии ядра.
